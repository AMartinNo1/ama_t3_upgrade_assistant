==============
Use Translations
==============

.. figure:: Images/localization-at-white-rounded@2x.png
   :alt: Crowdin Logo - (c) crowdin

   Crowdin Logo - (c) crowdin


With version 2.0.1 translations have moved to `crowdin <https://crowdin.com/project/typo3-extension-amat3upgradeas>`__ and
therefore will have to be fetched using the TYPO3 translation manager. The same core language files will be updated.
